# ■キーコンフィグについて
わかりづらいためキーコンフィグについて記します。

## LEFT の設定をしたい場合
* １．LEFT にカーソルを合わせます
* ２．エンターキー又はパッドの１番目のボタン（反応するやつが１番目のボタンです）を押します
* ３．割り当てたいキーを押します

以上の流れで LEFT に任意のキーを割り当てることができます、他のコンフィグも同様です。
